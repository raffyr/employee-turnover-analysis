# Análise de Turnover de Funcionários

![Capa do Projeto](capa_turnover.png)


## Objetivo

Este projeto tem como objetivo identificar os principais fatores associados ao turnover de funcioários por meio de análise exploratória de dados e Information Value (IV). O projeto foi desenvolvido como parte do meu portifólio em Análise de Dados.

## Ferramentas utilizadas

- Microsoft Excel
- Estatística Descritiva
- Tabelas Dinâmicas
- Information Value (IV)
- Análise Exploratória de Dados (EDA)

  ## Base de Dados

  A análise foi realizada utilizando uma base de dados de uma empresa do setor de tecnologia, contendo informações sobre os colaboradores, como idades, salário, horas extras, tempo de empresa, estado civil, frequência de viagens e turnover(desligamento).

  > **Observação:** A base de dados foi disponibilizada pela Preditiva – Escola de Dados para fins educacionais.

## Principais Resultados

- Colaboradores que realizavam horas extras apresentaram maior taxa de turnover.
- Faixas salariais mais baixas estiveram associadas a maiores índices de desligamento.
- Funcionários com menor tempo de empresa apresentaram maior probabilidade de saída.
- A faixa etária mais jovem também apresentou maior associação com o turnover.

  ## Recomendações

Com base na análise, foram propostas algumas recomendações para apoiar a tomada de decisão do setor de Recursos Humanos, como:

- Revisar a política de horas extras.
- Acompanhar colaboradores nos primeiros anos de empresa.
- Avaliar as faixas salariais com maior índice de desligamento.
- Desenvolver estratégias de retenção para os grupos com maior propensão ao turnover.
